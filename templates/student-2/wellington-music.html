Wellington Music is a web app designed to you keep you up to date with everything Wellington has to offer in music. From festivals to weekend venues, instrument stores to music schools, Wellington Music has you covered. It features some editor picks for the hottest spots right now, as well as constantly pulling live data to keep you in the know.

Wellington Music was built with VueJS with GooglePlaces API and Foursquare Places API integration. It includes some of the following features:

  1. Automatic route calculation and full route customisation through Leaflet Routing Machine.
  2. Dynamic calculations of all costs and statistics. Change your route, vehicle, or any other aspect of your trip and all calculations and costs are dynamically updated.
  3. JSON data stores everything you need to know about 12 vehicles. Seats, engine size, fuel economy, doors, boot size, and rental cost.
  4. Extensibility to import API data for car rental.
  5. UX and usability focussed.
